Cuerpo principal del contenido

La etiqueta main es en HTML5 una etiqueta semantica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la pagina. Unicamnete puede haber una sola etiqueta como esta en todo el documento.
El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezdos de pagina, los menues de navegacion y los pie de pagina). Es usado por buscadores para indexar.
Conviene usar aqui textos que ayuden a encontrar la pagina luego de procesos de indexacion de los buscadores.

Seccion etiquetas semanticas mas importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Las etiquetas semanticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

Reglas para etiquetas semanticas:

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION.
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. Osea un FOOTER podria anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento.
  • Reservar etiqueta h1 para usar dentro de etiquetas top level como main,header o footer.

  1. Meta title: Se encarga de informar a los motores de busqueda sobre cual es el titulo de una pagina web.
    Meta description: Se utiliza para describir brevemente el contenido de una pagina web.
  2. lang: Indica el idioma del texto mediante un codigo predefinido.
    dir: Indica la direccion del texto (para idiomas que escriben de derecha a izquierda).
    style: Establece de forma directa los estilos CSS de un elemento.
  3. La etiqueta header es usada en el cuerpo de la pagina, y puede ser usada multiples veces, por ejemplo, para determinar el principio de un articulo.
    En cambio, la etiqueta head es usada para metadata, como titulo, link, etc. y no se muestra en la pagina.
  4. UTF-8 o "8-bit UnicodeTransformationFormat" es una codificacion de caracteres que le asigna una cadena de bit determinada a cada caracter Unicode y que se puede leer como un numero binario. Esto significa que UFT-8 asigna un numero binario fijo a todas las consonantes, cifras y simbolos de un numero mayor de idiomas. En resumen, UTF-8 cubre casi todos los caracteres, los cuales tienen entre 1 y 4 bytes, mejorando la comunicacion internacional.
  5. Utilizamos el atributo "style" en HTML para agregar estilo a un elemento, como color, fuente , tamaño, y mas. Es universal, por lo tanto, es aplicable a cualquier etiqueta.
  6. Al insertar un enlace dentro de la etiqueta "" con el formato "href", podemos agregar el comando "target="_blank"" al final de la misma para indicar que queremos abrir el enlace en una pestaña nueva.
  7. Las etiquetas semanticas indican que es el contenido que contienen, en lugar de como se debe formatear al mostrar la pagina. Es decir, indican que son y cual es el significado de los elementos que contienen dentro. Con ellas se puede facilitar que cualquier maquina lea la pagina web HTML en un formato correspondiente. Algunos ejemplos son: section, article, footer, nav, etc.
  8. Los atributos basicos que aplicaria a una tabla, para conseguir un formato mas agradable a la vista, son:
    border para indicar si se dibujan bordes alrededor de las celdas y un borde alrededor de la tabla.
    bordercolor para definir el color del borde.
    colspan y rowspan para unir una celda con las celdas contiguas, tanto horizontal como verticalmente.
    background para colocar una imagen de fondo a la tabla.
    height y width para definir la altura y anchura en pixeles de una celda.
    bgcolor para darle un color de fondo a la tabla.
    align para alinear horizontalmente la tabla con el resto de la pagina.
  9. El body de HTTP es una parte opcional de un mensaje, que si esta incluido es usado para transportar la peticion o la respuesta. En otras palabras, transporta la informacion enviada.
    En cambio, el elemento body de HTML representa el contenido de un documento HTML. Solo puede haber un body por documento.
  10. Los atributos para agrupar celdas de una tabla son "colspan" de forma horizontal y "rowspan" de forma vertical, y son aplicados, de forma conjunta o individualmente, a la etiqueta "td"
